# Plain theme

Minimalist plain theme for vscode

## Installation

<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> on Windows/Linux → `ext install konstantin.plain`

or

<kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>P</kbd> on macOS → `ext install konstantin.plain`
